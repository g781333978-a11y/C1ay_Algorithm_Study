## 📖 项目简介 (Introduction)
本项目是我在 **考研备考 (408)** 及 **算法进阶** 过程中的代码归档仓库。
主要包含 LeetCode 刷题记录、408 数据结构真题、以及常用算法模板。

- **当前状态**: 🏃‍♂️ 进阶刷题中 (Phase 2)
- **编程语言**: C++ (主要)
- **开发环境**: VS Code (MacOS M1)

## 🗺️ 目录结构 (Structure)
本项目严格遵循知识点由浅入深的逻辑进行归档：

- `00_Templates/` - 竞赛/机试通用模板 (IO加速, 并查集, 线段树等)
- `01_Basics/` - 基础算法 (二分, 排序, 位运算)
- `02_Linear_DS/` - 线性数据结构 (链表, 栈, 队列)
- `03_Graph/` - 图论 (DFS, BFS, 最短路, MST)
- `04_Tree/` - 树形结构 (二叉树, 堆, Trie)
- `05_DP/` - 动态规划专题 (线性DP, 背包, 状压)
- `99_408_Real_Exam/` - 408 考研真题代码实现
